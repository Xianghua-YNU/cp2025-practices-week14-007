## 2. 理论背景

### 2.1 洛伦兹方程组

**方程形式：**
dx/dt = σ(y - x)
dy/dt = rx - y - xz  
dz/dt = xy - bz


**标准参数：**
- σ = 10.0
- r = 28.0
- b = 8/3

**初始条件：**
- 基准条件：(0.1, 0.1, 0.1)
- 扰动条件：(0.10001, 0.1, 0.1)

def lorenz_system(state, sigma=10.0, r=28.0, b=8/3):
    x, y, z = state
    dxdt = sigma * (y - x)
    dydt = r*x - y - x*z
    dzdt = x*y - b*z
    return [dxdt, dydt, dzdt]

# 求解方程
sol = solve_ivp(lorenz_system, [0, 50], [0.1, 0.1, 0.1], 
                t_eval=np.linspace(0, 50, 5000), method='RK45')

# 绘制3D吸引子
fig = plt.figure()
